# Bundled cheerful sounds

Four short MP3 previews (`burst`, `rain`, `fireworks`, `snow`) ship with **flutter_confetti_engine**. Use them via the exported **`ConfettiBundledSounds`** API in Dart.

**License:** [Mixkit Sound Effects Free License](https://mixkit.co/license/#sfxFree) — free for commercial and non-commercial use. Replace with your own assets if you prefer.

Your app **must** declare the assets in `pubspec.yaml` so Flutter bundles them:

```yaml
flutter:
  assets:
    - packages/flutter_confetti_engine/assets/sounds/
```
